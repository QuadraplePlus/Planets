지구와 달의 공전, 자전은 스크립트 하나로 제어하였으며 . 자전속도는 인스펙터 뷰에서 직접 입력 가능합니다 .공전속도는 자전속도에 일정 비례하여 따릅니다. 

또한 지구는 자전을 하면서 일정 주기로 밤과 낮이 바뀌며 , 이는 셰이더를 이용해 제어하였습니다.
